错排问题公式：
考虑一个有`n`个元素的排列，若一个排列中所有的元素都不在自己原来的位置上，那么这样的排列就称为原排列的一个错排。 `n`个元素的错排数记为$D_{n}$或$!n$。 研究一个排列错排个数的问题，叫做错排问题或称为更列问题。
$$derangementD_{n} = n!(1 - \frac{1}{1!} - \frac{1}{2!} - ... - \frac{1}{n!}) = (n-1)(D_{n-2} - D_{n-1})$$
可以推导出：
$$Q_{n} = D_{n}+D_{n-1}$$
错排规律：
$$D_n = n * D_{n-1} + (-1)^n$$
